# Portfolio Website – README

## Overview

This project is a personal portfolio website designed to present professional information, technical skills, and selected projects. The layout follows a clean, structured approach with a focus on readability and responsiveness.

---

## Website Sections

### Navigation

A fixed navigation bar at the top of the page that provides links to all major sections: Home, About, Skills, Projects, and Contact.

### Home (Hero Section)

Introduces the individual with a name and role description. Includes primary call-to-action buttons for quick navigation.

### Marquee

A horizontally scrolling section displaying key technologies and skills to add visual engagement.

### About

Provides a brief background, including education, interests, and focus areas.

### Skills

Displays technical competencies in a structured grid format for clarity and organization.

### Projects

Highlights selected work with project titles, descriptions, and associated technologies.

### Contact

Provides contact details, including an email address and relevant social links.

### Footer

Contains basic information and repeated navigation links for accessibility.

---

## Styling and Design

* Built using CSS with Flexbox and Grid for layout
* Dark theme with a consistent accent color
* Responsive design to support different screen sizes
